# Java Keywords

### **Access Modifiers**

- `public` → Accessible from anywhere.
- `protected` → Accessible within package + subclasses.
- `private` → Accessible only within the class.
- `default` (no keyword, package-private) → Accessible only within the same package.

### **Class & Object Related**

- `class` → Defines a class.
- `interface` → Defines an interface.
- `enum` → Defines an enumeration.
- `abstract` → Used to declare abstract classes/methods.
- `final` → Prevents inheritance, method overriding, or variable reassignment.
- `extends` → Inherits from a parent class.
- `implements` → Implements an interface.
- `new` → Creates new objects.
- `this` → Refers to current object.
- `super` → Refers to parent class object.

### **Control Flow**

- `if` → Conditional execution.
- `else` → Alternative branch in `if`.
- `switch` → Multiple conditional branches.
- `case` → Defines a branch in `switch`.
- `default` → Default branch in `switch`.
- `while` → Loop with condition checked first.
- `do` → Loop with condition checked after execution.
- `for` → Loop with initializer, condition, increment.
- `break` → Exits loop or `switch`.
- `continue` → Skips current iteration of loop.
- `return` → Exits method and optionally returns value.
- `yield` → Returns value from a `switch` expression.

### **Exception Handling**

- `try` → Block of code to test for exceptions.
- `catch` → Handles exceptions.
- `finally` → Block always executed after try-catch.
- `throw` → Used to throw an exception.
- `throws` → Declares exceptions a method may throw.

### **Data Types**

- `int` → 32-bit integer.
- `long` → 64-bit integer.
- `short` → 16-bit integer.
- `byte` → 8-bit integer.
- `char` → 16-bit Unicode character.
- `float` → 32-bit floating-point number.
- `double` → 64-bit floating-point number.
- `boolean` → Logical values `true/false`.
- `void` → No return type.

### **Modifiers & Storage**

- `static` → Belongs to class, not instance.
- `volatile` → Variable value is always read from main memory.
- `transient` → Prevents serialization of a field.
- `synchronized` → Ensures thread-safe access to a method/block.
- `native` → Specifies method implemented in non-Java code.
- `strictfp` → Forces floating-point calculations to follow IEEE standards.

### **Literals & Constants**

- `true` → Boolean true literal.
- `false` → Boolean false literal.
- `null` → Null reference literal.
- `const` → Reserved (not used).
- `goto` → Reserved (not used).

### **Packages & Imports**

- `package` → Defines a package.
- `import` → Imports a class or package.

### **Instance & Inheritance**

- `instanceof` → Tests if object is instance of a class/interface.
- `assert` → Used for debugging, checking conditions at runtime.
